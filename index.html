CGPA Calculator

How to use the calculator

  • Enter the number of courses you have completed so far.
  • Click Submit.
  • For each course, enter the number of credits* and the corresponding grade earned.
  • Click Calculate and your CGPA will be displayed below the button.
  • Additionally, the “+” button appends a row to the last row and the “–” deletes the last row.
* 3 credits for normal courses and 4 credits for courses with lab classes.

Note that this calculator conforms to the grade points as assigned for Independent University, Bangladesh. If your university has a different set of grade points, send me an email (armunalam@gmail.com), I'll update it.
